# SLAM导航控制服务

> 提供机器人系统 SLAM（同时定位与地图构建）和导航控制服务，通过 `SlamNavController` 可以通过 RPC 与话题订阅方式实现建图、定位、导航、地图管理与里程计数据获取。

## 接口定义
`SlamNavController` 是面向 SLAM 和导航控制的控制器，支持建图、定位、导航、地图管理与里程计相关控制操作，封装底层细节以供上层系统调用。
